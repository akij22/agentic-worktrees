Concept 5 · Minimal chrome

Header switcher + command palette

No persistent repository sidebar. The current repo sits in a header switcher; worktrees and sessions fill the full width. Press ⌘K (or click the search field) for a command palette that jumps across repositories, worktrees, and sessions. Best when one repository at a time is the common case.

Strengths

  • Maximum canvas for worktrees, sessions, and chat
  • Fast keyboard-driven multi-repo work without permanent lists
  • Clear “you are in this repository” via the header switcher
  • Command palette doubles as discovery + actions

Trade-offs

  • Cross-repo awareness is weaker until you open the palette
  • Mouse-only users pay an extra click to switch repos
  • Status across many repositories is not glanceable
  • Needs excellent empty states and recent-items ranking
